¡Bienvenid@ a aprende más!

Esta página tiene como objetivo ayudar con la organización, comunicación y control de tareas, avisos y alumnos

A continuación se enlistarán puntos importantes que se deberían seguir:

1. Los avisos son visibles para cualquier persona con acceso a la web, deben utilizarse para avisos globales.

2. Para agregar un alumno, primero se debe agregar un grupo al cual pueda ser asignado.

3. Para agregar una tarea, primero se debe agregar un grupo al cual pueda ser asignado.

4. Para eliminar algún elemento, se puede hacer doble clic sobre este (aplica para tareas, alumnos y avisos).

5. Si encuentra algún problema con la aplicación (se congela la pantalla, los datos no se actualizan, entre otros) intente reiniciar la página.

6. Si tiene más dudas sobre el funcionamiento, puede consultar el manual de maestro que se proporciona en el siguiente enlace: Link Manual Tutor

Esta página tiene como objetivo ayudar con la organización, comunicación y control de información que puede ser relevante para usted

A continuación se enlistarán puntos importantes que se deberían seguir:

1. Solo se visualizarán las tareas correspondientes al grupo seleccionado al iniciar sesión, si se desea cambiar el grupo, puede dirigirse a "Configuración"

2. Por seguridad, no se recomienda cambiar demasiado de grupo, solo debe realizarse en caso de no haber recordado o no conocer el grupo del alumno.

3. Actualmente se encuentra en una cuenta de tutor, por lo que solo puede visualizar las tareas asignadas, vencidas y los avisos.

4. Los avisos son planeados para todo el alumnado, espere ver ahí comunicados relacionados a juntas, entrega de documentos, entre otros.
